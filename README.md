# WEB会議の前のカメラとマイクテストの自動化

Power Automate Desktopを使って、カメラを起動し、サウンドのマイクテストを実行します。

## セットアップ～インストール～スタート

リポジトリをクローンします

git clone https://github.com/Junk-Programing/web-test.git

このフォルダを、Power Automate Desktopのフローが保存されているフォルダに貼り付けます。

(例) C:\Users\ユーザー名\AppData\Local\Microsoft\Power Automate Desktop\Console\Workspace

このファイルを元に、ChatGPT等に解析させて、自分のPower Automateのフロー作成に使用してください。

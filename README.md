# WEB会議の前のカメラとマイクテストの自動化

Power Automate Desktopを使って、カメラを起動し、サウンドのマイクテストを実行します。これにより、WEB会議前のカメラとマイクテストを実行できます。

## セットアップ～インストール～スタート

リポジトリをクローンします

git clone https://github.com/Junk-Programing/web-test.git

このフォルダを、Power Automate Desktopのフローが保存されているフォルダに貼り付けます。

フォルダ例
C:\Users\ユーザー名\AppData\Local\Microsoft\Power Automate Desktop\Console\Workspace

Power Automate Desktopを起動し、フローを実行します。フローはショートカットによる起動も可能です。
